# Security

This project is a static portfolio website.

It does not require a backend server, database, authentication, or user accounts.

## Public data

The website only loads public data from GitHub:

- public repositories;
- `.portfolio.json`;
- `README.md`;
- repository metadata from the GitHub API.

## Secrets

Do not commit secrets, tokens, API keys, private configuration files, or
credentials to this repository.
