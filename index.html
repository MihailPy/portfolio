How it works

Сайт автоматически собирает проекты из GitHub без ручного списка в коде.

01

Repository topic

Проект попадает на сайт, если у репозитория есть topic portfolio.

02

Project metadata

Карточка проекта строится из файла .portfolio.json.

03

README preview

Подробное описание загружается из README.md выбранного репозитория.

Project format

Каждый проект в портфолио описывается отдельным файлом в репозитории.

{
    "title": "Project title",
    "status": "ready",
    "category": "CLI tool",
    "priority": 1,
    "stack": ["Python", "pytest"],
    "summary": "Short project description.",
    "highlights": [
      "Main feature",
      "Architecture detail",
      "Testing or deployment detail"
    ]
  }
